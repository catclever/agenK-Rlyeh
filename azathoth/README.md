# @agent-k/azathoth

## 模块简介

`azathoth` 是 Agent K 系统中的**原子组件管理器 (Atom Manager)** 与**组件注册表 (Atom Registry)**。

在克苏鲁神话中，阿撒托斯（Azathoth）是盲目痴愚之神，也是宇宙的源头与核心。在本项目中，它代表着所有 UI 元素的原始素材库。

## 当前状态

目前该模块处于**早期原型/占位阶段**。
- **UI 占位**：提供了一个暗色系的 React 面板界面。
- **功能预期**：未来将用于对接外部的 `/atoms` 目录或组件库，通过解析 JSON Schema 动态管理和下发无状态的“原子级” UI 组件，支持拖拽式设计系统的运转。
- **依赖状态**：等待与真正的组件解析逻辑连接，当前“已加载组件”列表为硬编码的空状态。
